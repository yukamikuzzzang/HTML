What is HTML?

HTML은 무엇인가요?

HTML stands for Hyper Text Markup Language

HTML은 하이퍼 텍스트 마크업 언어를 의미합니다.

HTML is the standard markup language for creating Web Pages

HTML은 웹페이지를 만들기 위한 표준 언어입니다.

HTML describes the structure of a Web page

HTML은 웹페이지의 구조를 보여줍니다.

HTML consists of a series of elements

HTML은 일련의 요소들로 구성됩니다.

HTML elements tell the browser how to display the content

HTML은 브라우저에게 어떻게 내용을 보여줄지를 말해줍니다.

HTML elements label pieces of content such as "this is a heading","this is a paragraph", "this is link", etc.

HTML 요소들은 "이것은 제목이다", "이것은 문단이다", "이것은 링크다"와 같은 콘텐츠의 라벨을 붙입니다.
